# QuantifyingDeliveryLabor 
지형 특성에 따른 배송 노동 환경 정량화 시뮬레이터 
# 프로젝트 소개
택배 배달원은 건당 동일한 수수료를 받지만, 배송 구역의 경사도와 골목 너비에 따라 실제 소요 시간과 체력 소모가 크게 달라집니다.
본 프로젝트에서는 도로망위에서 경사도 ,도로폭 데이터를 실시가능로 반영하여 지형 조건이 배달 노농에 미치는 영향을 시각화합니다. 
관악구 17개 동의 실제 도로 데이터(DEM 경사도 + 행안부 도로폭)를 기반으로 배달원 에이전트를 시뮬레이션하여, "지형이 만드는 배송 불평등" 을 데이터로 증명합니다.
